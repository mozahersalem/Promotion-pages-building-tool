# Promotion-pages-building-tool

# Landing Page Template


### Install

```
    $ npm install 
    Mac and linux users use sudo if needed
    $ sudo npm install

```




### DEV Mode

```
    $ grunt 

    $ grunt serve 
    will run localhost:9000 - with auto reload

```

### tasks

``` 
    grunt
    grunt serve
    grunt build
    grunt serve-bild
    grunt upload
    grunt zip
    grunt clean // use with caution 

```

### Build Mode


```
    $ grunt bulid

    $ grunt serve-build  
    will run localhost:8000 - NO auto reload

```
